# Finance Dashboard

A simple React-based finance dashboard to track your income and expenses in real time.

## Features
- **Summary Cards**: View Total Balance, Total Income, and Total Expenses at a glance.
- **Transaction List**: Add, view, and filter transactions by type (Income/Expense).
- **Charts**: Visualize spending breakdown by category and track your balance over time.
- **Filtering**: Easily filter transactions to focus on specific types.

## Tech Stack
- **Frontend**: React, Vite
- **Styling**: CSS (Vanilla)
- **Charting**: Any chart library you integrated (e.g., Chart.js, Recharts)
